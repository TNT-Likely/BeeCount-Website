# Calendar

The calendar feature lets you intuitively view daily income and expense, quickly understanding your spending patterns.

## Access Calendar

1. Tap "Statistics" in the bottom navigation
2. Tap "Calendar" tab at the top

## Calendar View

The calendar displays in monthly view, showing below each date:

- **Income Amount** - Green/Red display (depends on color scheme setting)
- **Expense Amount** - Red/Green display (depends on color scheme setting)
- **Net Amount** - Income minus expense

Color intensity indicates amount size, spotting spending peaks at a glance.

## View Details

Tap any date to view all transactions for that day:

- Transaction list sorted by time
- Category icons and amounts
- Notes information

## Record Directly on the Selected Date

Since 3.2.3, after you select a date on the calendar, a "**Record on this day**" entry appears above the transaction list. Tap it to jump straight into the entry form with the date pre-filled — no more switching to the record page and manually picking the day.

- Great for backfilling past days: flip to a date, see if anything is missing, tap and add
- Date / month switches now use skeleton placeholders so the page no longer flickers on reload

## Quick Navigation

- **Swipe Left/Right** - Switch months
- **Jump to Today** - Tap "Today" button in top right
- **Select Month** - Tap month title at top

## Use Cases

- 📊 **Spending Pattern Analysis** - See which dates have higher spending
- 📅 **Bill Date Reminders** - Mark fixed expense dates
- 💰 **Income Cycles** - View salary payment dates
- 🎯 **Spending Control** - Visually see daily expenses

## Works with Statistics

Calendar complements statistics charts:

- Statistics page shows trends and proportions
- Calendar page shows specific date details
- Use together for comprehensive financial overview
