# Account Management

BeeCount supports multiple accounts, each with independent balance tracking.

## Account Types

- **Cash** - Physical cash on hand
- **Savings** - Bank savings accounts
- **Credit Card** - Credit card accounts
- **Alipay/WeChat** - E-wallets
- **Investment** - Stocks, funds, etc.

## Create an Account

1. Go to "Me" → "Account Management"
2. Tap "+" in the top right to add an account
3. Select type, enter name and initial balance
4. Optionally set account icon and color

## Account Balance

When recording transactions, the system automatically updates account balances:

- Expense → Account balance decreases
- Income → Account balance increases
- Transfer → Source decreases, destination increases

## Account Details

Tap an account to view:

- Current balance
- Transaction history
- Statistics charts

## Default Account

You can set default accounts for income and expenses, which will be auto-selected when recording.
