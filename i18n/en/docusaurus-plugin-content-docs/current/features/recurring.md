# Recurring Entries

Automatically record fixed income and expenses - ideal for salary, rent, subscriptions, and other periodic transactions.

## Create a Recurring Entry

1. Go to "Discover" â†’ "Recurring Entries"
2. Tap "+" to create a new recurring entry
3. Fill in transaction details (amount, category, account, etc.)
4. Set the frequency and execution time

## Frequency Types

- **Daily** - Records automatically every day
- **Weekly** - On a fixed day each week
- **Monthly** - On a fixed date each month
- **Yearly** - On a fixed date each year

## Use Cases

- **Salary** - Monthly income on a fixed date
- **Rent** - Fixed monthly expense
- **Subscriptions** - Video streaming, music services, etc.
- **Phone Bill** - Monthly recurring charges

## Managing Recurring Entries

- Pause, resume, or delete recurring entries
- Set start and end dates
