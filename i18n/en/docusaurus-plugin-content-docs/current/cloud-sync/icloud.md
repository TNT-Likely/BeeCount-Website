# iCloud Sync

The preferred sync method for iOS users - zero configuration, works out of the box.

## Advantages

- ✅ **Zero Config** - No account registration or setup needed
- ✅ **Auto Sync** - Syncs automatically in the background
- ✅ **Multi-device** - Syncs across iPhone and iPad
- ✅ **Secure** - Uses Apple's end-to-end encryption
- ✅ **Free** - Uses your own iCloud storage

## Enable Sync

1. Ensure your device is signed into iCloud
2. Go to "Me" → "Cloud Service"
3. Select "iCloud"
4. Enable sync

## How It Works

Data is stored in your iCloud Drive:

```
iCloud Drive/
└── BeeCount/
    └── Data files
```

## Multi-device Sync

Sign in with the same Apple ID on all devices, and data will sync automatically.

## Notes

- iOS/iPadOS only
- Requires sufficient iCloud storage space
- Initial sync may take a few minutes

## FAQ

**Q: Sync not working?**

A: Check if iCloud is signed in, verify network connection, try manually triggering sync.

**Q: How to restore data on a new phone?**

A: Sign in with the same Apple ID on your new phone and enable iCloud sync - data will restore automatically.
