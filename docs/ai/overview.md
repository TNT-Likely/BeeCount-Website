# AI 助手概览

蜜蜂记账集成 AI 功能，让记账更智能、更轻松。

## 功能一览

| 功能 | 说明 |
|------|------|
| [语音记账](./voice) | 说一句话完成记账 |
| [图片识别](./image) | 拍照自动识别账单 |
| [AI 对话](./chat) | 智能财务助手 |

## 支持的 AI 服务

- **智谱 GLM** - 国内可用，推荐
- **自定义服务商** - 支持任意兼容 OpenAI 协议的服务商
- **本地模型** - 离线可用（开发中）

## 模型选择

智谱 GLM 提供多种模型可选：

| 模型 | 特点 | 推荐场景 |
|------|------|----------|
| GLM-4-Flash | 免费，速度快 | 日常记账，推荐 |
| GLM-4-Air | 平衡性能和成本 | 需要更好理解能力时 |
| GLM-4-AirX | 更强推理能力 | 复杂场景 |
| GLM-4-FlashX | 增强版免费模型 | 免费但需更好效果时 |
| GLM-4-Plus | 旗舰模型 | 追求最佳效果 |
| GLM-4-Long | 超长上下文 | 长对话场景 |

> 推荐使用 **GLM-4-Flash**，免费且满足日常记账需求。

## 配置 AI

1. 进入「我的」→「智能记账」→「AI 小助手」，或「发现」→「常用功能」→「AI 设置」
2. 选择 AI 服务提供商
3. 输入 API Key
4. 保存配置

## 自定义 AI 服务商

除了内置的智谱 GLM，你还可以添加任意兼容 OpenAI 协议的服务商：

### 支持的服务商

- **硅基流动 (SiliconFlow)** - 国内可用，价格实惠
- **DeepSeek** - 国产大模型，性价比高
- **OpenAI** - 需要海外网络环境
- **其他兼容服务** - 任何提供 OpenAI 兼容 API 的服务

### 添加自定义服务商

1. 进入「我的」→「智能记账」→「AI 小助手」
2. 点击「添加服务商」
3. 填写服务商信息：
   - **名称** - 自定义名称，方便识别
   - **API 地址** - 服务商提供的 API 端点
   - **API Key** - 你的 API 密钥
   - **模型名称** - 根据需要配置文本/图片/语音模型

### 能力绑定

你可以为不同的 AI 能力绑定不同的服务商：

| 能力 | 说明 | 示例 |
|------|------|------|
| 文本理解 | 语音转账单、AI 对话 | 使用 DeepSeek |
| 图片识别 | 拍照识别账单 | 使用智谱 GLM |
| 语音识别 | 语音转文字 | 使用智谱 GLM |

这样你可以根据各服务商的优势，灵活组合获得最佳体验。

## 获取 API Key

### 智谱 AI

<video width="100%" controls>

  您的浏览器不支持视频播放

1. 访问 [智谱开放平台](https://open.bigmodel.cn/)
2. 注册账号
3. 进入「API 密钥」页面，创建新的 API Key
4. 复制 API Key 到蜜蜂记账

> 💡 新用户注册后通常有免费额度可用，GLM-4-Flash 模型完全免费。

### 硅基流动

1. 访问 [硅基流动](https://siliconflow.cn/)
2. 注册账号并完成实名认证
3. 在控制台获取 API Key
4. API 地址填写：`https://api.siliconflow.cn/v1`

### DeepSeek

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 注册账号
3. 创建 API Key
4. API 地址填写：`https://api.deepseek.com/v1`
