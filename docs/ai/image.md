# 图片识别

拍照或选择图片，AI 自动识别账单信息。

![图片识别](/img/preview/zh/13-ocr-recognition.png)

## 使用方式

1. 进入记账页面
2. 点击相机图标
3. 拍照或从相册选择
4. AI 自动识别账单
5. 确认并保存

## 支持的场景

- **购物小票** - 超市、商场收据
- **外卖订单** - 美团、饿了么截图
- **转账记录** - 微信、支付宝转账截图
- **发票** - 电子发票、纸质发票

## 识别内容

AI 会尝试识别：

- 金额
- 商家名称
- 消费时间
- 消费项目

## 一次识别多笔

3.2.3 起,手机端 AI 也支持**一张图里识别多笔交易**(原本只取第一笔)。账单截图、超市小票、支付宝/微信月度账单等多行场景,识别后会弹出 N 笔交易草稿:

- 逐笔确认/调整金额、分类、账户、标签、备注
- 整张原图作为附件共享挂在所有 N 笔交易上,日后任意一笔点附件都能看到原图
- 任意一笔点「跳过」即可丢弃,只保存想要的那几笔

## 使用技巧

- 图片清晰，文字可辨认
- 尽量包含完整的金额信息
- 避免强光反射或阴影

## 手动调整

AI 识别结果可能不完全准确，请在保存前确认并手动调整。

## 自动标签

使用图片记账时，系统会自动为交易添加「图片记账」标签，方便后续筛选和统计。

## 自动保存附件

识别用的图片会自动保存为交易附件，无需手动添加。你可以在交易详情中查看原始图片。

## Web 端图片记账

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端后,桌面浏览器可以**直接粘贴图片**:

1. 任意页面按 **⌘K / Ctrl+K** 打开命令面板
2. **⌘V / Ctrl+V** 粘贴一张图(剪贴板里的截图 / 拖拽到面板里也可)
3. 默认动作切到「AI 记账(图片)」→ Enter
4. 弹出 N 笔交易草稿,可逐笔编辑分类 / 账户 / 标签 / 备注
5. 点「保存所选」批量入库

服务器端用 vision LLM(智谱 GLM-4V Flash 等,需在 [AI 配置](./overview.md#web-端-ai-配置) 里绑定)解析,**N 笔交易共享同一张原图作为附件**。失败会显示 LLM 原始输出,方便排查。

跟手机端一致,识别完成的交易会自动加「AI 记账」+「图片记账」标签。
