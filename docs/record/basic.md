# 基础记账

蜜蜂记账支持三种交易类型：支出、收入和转账。

## 快速记账

点击底部中央的 **「+」按钮** 进入记账页面。

![记账页面](/img/preview/zh/03-edit-transaction.png)

### 操作步骤

1. **选择类型** - 支出 / 收入 / 转账
2. **输入金额** - 使用数字键盘输入
3. **选择分类** - 点击分类图标选择
4. **选择账户** - 选择关联账户
5. **完成** - 点击右上角保存

> ✨ **v3.3.0 起**:记账键盘支持加减乘除——长按或双击数字键盘可切换出 + − × ÷,AA 分摊、合并小票时直接在键盘上算。

### 可选信息

- **备注** - 添加交易说明
- **标签** - 打上一个或多个标签
- **日期时间** - 修改记账时间（补记）
- **图片** - 添加消费凭证

## 编辑与删除

- **点击** 交易记录可编辑
- **左滑** 交易记录可删除

## 转账记录

转账用于在不同账户间转移资金：

- 选择「转账」类型
- 选择转出账户和转入账户
- 输入转账金额

系统会自动更新两个账户的余额。

## Web 端记账

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端后,桌面浏览器记账更高效:

- **+ 新建交易按钮** — 交易页右上角点击,弹窗填字段保存
- **⌘K / Ctrl+K 命令面板** — 任意页面按快捷键 → 输入金额或备注 → 选「新建交易」直接记
- **批量操作** — 列表页支持多选删除 / 导出 CSV(详见[交易管理](../features/record.md))
- **桌面端键盘体验** — 数字键盘、Tab 切换字段、Enter 保存

Web 端创建的交易通过云同步秒级推到手机,反之亦然。
