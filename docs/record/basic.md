# 基础记账

蜜蜂记账支持三种交易类型：支出、收入和转账。

## 快速记账

点击底部中央的 **「+」按钮** 进入记账页面。

![记账页面](/img/preview/zh/03-edit-transaction.png)

### 操作步骤

1. **选择类型** - 支出 / 收入 / 转账
2. **输入金额** - 使用数字键盘输入
3. **选择分类** - 点击分类图标选择
4. **选择账户** - 选择关联账户
5. **完成** - 点击右上角保存

### 可选信息

- **备注** - 添加交易说明
- **标签** - 打上一个或多个标签
- **日期时间** - 修改记账时间（补记）
- **图片** - 添加消费凭证

## 编辑与删除

- **点击** 交易记录可编辑
- **左滑** 交易记录可删除

## 转账记录

转账用于在不同账户间转移资金：

- 选择「转账」类型
- 选择转出账户和转入账户
- 输入转账金额

系统会自动更新两个账户的余额。
