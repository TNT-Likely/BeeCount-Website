# 周期记账

自动记录固定收支，适合工资、房租、订阅等周期性交易。

## 创建周期账单

1. 进入「发现」→「周期记账」
2. 点击「+」创建
3. 填写交易信息
4. 设置周期和执行时间

## 周期类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 每日 | 每天执行 | 日常通勤 |
| 每周 | 每周固定日 | 周末娱乐 |
| 每月 | 每月固定日 | 工资、房租 |
| 每年 | 每年固定日 | 年费会员 |

## 常见场景

- **工资** - 每月固定日期收入
- **房租/房贷** - 每月固定支出
- **订阅服务** - 视频/音乐会员
- **话费** - 每月固定扣费
- **保险** - 年度保费

## 管理周期账单

- **暂停** - 临时停止执行
- **恢复** - 重新启用
- **删除** - 完全移除
- **设置期限** - 指定开始和结束日期
