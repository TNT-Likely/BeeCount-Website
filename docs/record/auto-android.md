# Android 自动记账

Android 支持多种自动记账方式：截图识别、分享记账、通知监听。

## 方式一：截图记账

支付完成后截图，蜜蜂记账自动识别。

### 使用步骤

1. 完成支付后截图
2. 蜜蜂记账检测到新截图
3. 自动识别金额、商家信息
4. 弹出确认对话框
5. 确认后完成记账

### 开启截图监听

1. 打开蜜蜂记账
2. 进入「我的」然后点击「自动记账」
3. 开启「截图自动识别」
4. 授予存储权限

### 支持的截图类型

- 微信支付成功页
- 支付宝支付成功页
- 银行 App 交易详情
- 其他支付凭证截图

## 方式二：分享记账

将支付信息分享给蜜蜂记账。

### 使用步骤

1. 在支付 App 中找到交易记录
2. 点击「分享」按钮
3. 选择「蜜蜂记账」
4. 自动识别并填充信息
5. 确认后完成记账

### 支持的分享内容

- 文本（包含金额的消费描述）
- 图片（支付截图）
- 链接（交易详情页）

## 方式三：通知监听

监听支付通知，自动记账。

### 配置步骤

1. 进入「我的」然后点击「自动记账」
2. 开启「通知监听」
3. 授予通知访问权限
4. 选择要监听的 App（微信、支付宝等）

### 工作原理

1. 监听指定 App 的通知
2. 识别支付成功通知
3. 提取金额和商家信息
4. 自动创建记账记录

### 注意事项

- 需要授予「通知使用权」
- 部分系统可能限制后台运行
- 建议加入电池优化白名单

## 自动识别能力

蜜蜂记账使用 AI 识别以下信息：

- 金额：95% 以上准确率
- 商家名称：90% 以上准确率
- 支付方式：85% 以上准确率
- 消费分类：80% 以上准确率

## 配置建议

### 最佳实践

1. 开启截图识别 - 最稳定可靠
2. 授予必要权限 - 存储、通知
3. 加入白名单 - 防止被系统杀死
4. 定期检查 - 确认自动记账正常工作

### 省电设置

如果担心耗电，可以：
- 只开启截图识别（被动触发）
- 关闭通知监听（主动轮询）

## 常见问题

**截图识别没有反应？**

1. 检查是否授予存储权限
2. 检查是否开启了截图监听
3. 尝试重启应用

**识别结果不准确？**

AI 识别可能有误差，支持手动修改后保存。识别准确率会随着使用不断提升。

**后台被杀死？**

1. 将蜜蜂记账加入电池优化白名单
2. 允许后台运行
3. 锁定最近任务中的应用
