# 长按快捷操作

长按蜜蜂记账应用图标，可快速启动常用功能，无需打开 App 再查找。

## 支持平台

- iOS（iPhone 6s 及以上，需 iOS 13+）
- Android（Android 7.1 及以上）

## 快捷操作

长按应用图标后，会显示以下快捷菜单：

| 操作 | 说明 |
|------|------|
| 图片记账 | 从相册选择图片，AI 自动识别账单 |
| 拍照记账 | 打开相机拍照，AI 自动识别账单 |
| 语音记账 | 语音输入，AI 自动解析记账 |
| AI 小助手 | 打开 AI 对话，智能财务助手 |

## 使用方式

### iOS

1. 在主屏幕找到蜜蜂记账图标
2. 长按图标约 1 秒
3. 弹出快捷菜单后，点击需要的操作

### Android

1. 在桌面或应用抽屉找到蜜蜂记账图标
2. 长按图标
3. 弹出快捷菜单后，点击需要的操作

> 💡 部分 Android 启动器可能不支持此功能，请使用系统自带启动器或主流第三方启动器。

## 使用场景

- **收到账单短信**：长按图标 → 图片记账 → 截图识别
- **线下消费**：长按图标 → 拍照记账 → 对准小票拍照
- **快速记一笔**：长按图标 → 语音记账 → 说「午餐 35 元」
- **查询财务问题**：长按图标 → AI 小助手 → 对话咨询

## Web 端快捷入口

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端后,**⌘K / Ctrl+K** 命令面板是桌面端的"长按快捷"等价物:

### AI 截图记账(粘贴图片)

任意页面按 ⌘K → **直接 ⌘V / Ctrl+V 粘贴一张图**(支付宝 / 微信支付凭证、信用卡账单截图、银行流水截图等)→ 默认动作切到「AI 记账(图片)」→ 回车 → 弹出 N 笔 tx draft 编辑确认 → 批量保存。

服务器端用 vision LLM(智谱 GLM-4V Flash 等)解析,N 笔交易共享同一张原图作为附件。

### AI 文字记账(粘贴文字)

⌘K → 粘贴一段文字(微信账单文本、Excel 一段表格、自然语言描述如"昨天打车 30 块 + 午餐 25")→ 默认动作切到「AI 记账(文字)」→ 回车 → tx drafts 编辑确认。

详见 [AI 智能记账](../ai/image.md) / [AI 文字记账](../ai/chat.md)。

### 命令面板其它快捷

- 输入关键词 → 搜索交易(默认动作)
- 输入 `?xxx` → 问 AI 文档站
- 选「新建交易」/「年度报告」/「导出本月 CSV」/「导出本年 CSV」/「导入账本数据」等高频动作
- 跳到任意账本 / 页面
