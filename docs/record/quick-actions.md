# 长按快捷操作

长按蜜蜂记账应用图标，可快速启动常用功能，无需打开 App 再查找。

## 支持平台

- iOS（iPhone 6s 及以上，需 iOS 13+）
- Android（Android 7.1 及以上）

## 快捷操作

长按应用图标后，会显示以下快捷菜单：

| 操作 | 说明 |
|------|------|
| 图片记账 | 从相册选择图片，AI 自动识别账单 |
| 拍照记账 | 打开相机拍照，AI 自动识别账单 |
| 语音记账 | 语音输入，AI 自动解析记账 |
| AI 小助手 | 打开 AI 对话，智能财务助手 |

## 使用方式

### iOS

1. 在主屏幕找到蜜蜂记账图标
2. 长按图标约 1 秒
3. 弹出快捷菜单后，点击需要的操作

### Android

1. 在桌面或应用抽屉找到蜜蜂记账图标
2. 长按图标
3. 弹出快捷菜单后，点击需要的操作

> 💡 部分 Android 启动器可能不支持此功能，请使用系统自带启动器或主流第三方启动器。

## 使用场景

- **收到账单短信**：长按图标 → 图片记账 → 截图识别
- **线下消费**：长按图标 → 拍照记账 → 对准小票拍照
- **快速记一笔**：长按图标 → 语音记账 → 说「午餐 35 元」
- **查询财务问题**：长按图标 → AI 小助手 → 对话咨询
