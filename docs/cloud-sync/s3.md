# S3 协议同步

支持 S3 兼容的对象存储服务。

## 支持的服务

- **Cloudflare R2** - 推荐，10GB 免费
- **AWS S3** - 亚马逊云存储
- **MinIO** - 开源自建
- **阿里云 OSS** - S3 兼容模式

## Cloudflare R2 配置（推荐）

### 1. 创建 R2 存储桶

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 进入「R2」
3. 点击「Create bucket」
4. 输入存储桶名称

### 2. 创建 API Token

1. 进入「R2」→「Manage R2 API Tokens」
2. 点击「Create API token」
3. 权限选择「Object Read & Write」
4. 记录 Access Key ID 和 Secret Access Key

### 3. 配置蜜蜂记账

1. 进入「我的」→「云服务」
2. 选择「S3」
3. 填写配置：
   - Endpoint: `https://<account_id>.r2.cloudflarestorage.com`
   - Access Key ID: 上一步获取
   - Secret Access Key: 上一步获取
   - Bucket: 存储桶名称
   - Region: `auto`
4. 保存并测试

## AWS S3 配置

1. 登录 AWS Console
2. 创建 S3 存储桶
3. 创建 IAM 用户并获取密钥
4. 在蜜蜂记账中配置

## MinIO 自建

适合有服务器的技术用户，可以完全自主部署。

```bash
docker run -p 9000:9000 -p 9001:9001 minio/minio server /data --console-address ":9001"
```
