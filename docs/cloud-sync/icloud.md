# iCloud 同步

iOS 用户首选的云同步方式，零配置开箱即用。

## 优势

- ✅ **零配置** - 无需注册账号，无需配置
- ✅ **自动同步** - 后台自动同步
- ✅ **多设备** - iPhone、iPad 自动同步
- ✅ **安全** - 使用 Apple 的端到端加密
- ✅ **免费** - 使用你自己的 iCloud 存储

## 开启同步

1. 确保设备已登录 iCloud
2. 进入「我的」→「云服务」
3. 选择「iCloud」
4. 开启同步

## 同步原理

数据存储在你的 iCloud Drive 中：

```
iCloud Drive/
└── BeeCount/
    └── 数据文件
```

## 多设备同步

在所有设备上使用同一 Apple ID 登录 iCloud，数据会自动同步。

## 注意事项

- 仅 iOS/iPadOS 支持
- 需要 iCloud 有足够存储空间
- 首次同步可能需要几分钟

## 常见问题

**Q: 同步不生效？**

A: 检查 iCloud 是否登录，检查网络连接，尝试手动触发同步。

**Q: 换新手机如何恢复数据？**

A: 新手机登录同一 Apple ID，开启 iCloud 同步即可自动恢复。
