# WebDAV 同步

使用 WebDAV 协议同步数据，支持坚果云、群晖等。

## 支持的服务

- **坚果云** - 国内推荐
- **群晖 NAS** - 自建服务
- **NextCloud** - 开源自建
- **其他 WebDAV 服务**

## 坚果云配置

### 1. 注册坚果云

访问 [坚果云](https://www.jianguoyun.com/) 注册账号。

### 2. 获取应用密码

1. 登录坚果云网页版
2. 进入「账户信息」→「安全选项」
3. 添加应用密码
4. 记录生成的密码

### 3. 配置蜜蜂记账

1. 进入「我的」→「云服务」
2. 选择「WebDAV」
3. 填写配置：
   - 服务器：`https://dav.jianguoyun.com/dav/`
   - 用户名：坚果云账号（邮箱）
   - 密码：应用密码（非登录密码）
4. 保存并测试

## 群晖 NAS 配置

### 1. 启用 WebDAV

1. 打开群晖 DSM
2. 进入「套件中心」安装 WebDAV Server
3. 启用 WebDAV 服务

### 2. 配置蜜蜂记账

- 服务器：`http://你的NAS地址:5005/`
- 用户名：NAS 用户名
- 密码：NAS 密码

## 同步目录

数据会保存在 WebDAV 根目录的 `/BeeCount/` 文件夹中。
