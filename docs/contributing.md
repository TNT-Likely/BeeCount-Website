# 参与贡献

蜜蜂记账是一个开源项目，欢迎你参与贡献！

## 贡献方式

### 📝 完善文档

文档对用户非常重要，你可以帮助：

- **修正错误** - 发现文档错误或过时内容
- **补充说明** - 添加更详细的使用说明
- **添加截图** - 补充功能截图
- **翻译** - 帮助翻译英文文档

#### 如何贡献文档？

1. Fork [BeeCount-Website](https://github.com/TNT-Likely/BeeCount-Website) 仓库
2. 修改 `docs/` 目录下的 Markdown 文件
3. 提交 Pull Request

文档使用 Markdown 格式，非常简单！

### 🐛 报告问题

发现 Bug 或有功能建议？

1. 访问 [GitHub Issues](https://github.com/TNT-Likely/BeeCount/issues)
2. 点击「New Issue」
3. 选择合适的模板
4. 详细描述问题

好的 Issue 应该包含：

- 问题描述
- 复现步骤
- 期望行为
- 实际行为
- 设备信息（型号、系统版本）
- 截图（如有）

### 💻 贡献代码

如果你会 Flutter 开发：

1. Fork [BeeCount](https://github.com/TNT-Likely/BeeCount) 仓库
2. 创建 feature 分支
3. 提交代码
4. 创建 Pull Request

#### 开发环境

- Flutter 3.27+
- Dart 3.6+
- Xcode（iOS 开发）
- Android Studio（Android 开发）

### ⭐ Star 项目

最简单的支持方式 - 给项目一个 Star！

[GitHub - BeeCount](https://github.com/TNT-Likely/BeeCount)

### 💬 参与讨论

加入社区，和其他用户交流：

- [GitHub Discussions](https://github.com/TNT-Likely/BeeCount/discussions)
