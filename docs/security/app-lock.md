# 应用锁

蜜蜂记账支持应用锁功能，通过 PIN 码和生物识别保护你的财务隐私。

## 功能介绍

- **PIN 码锁定** - 设置 4 位 PIN 码，每次打开应用时需要验证
- **生物识别** - 支持指纹识别和面容识别（Face ID），快速解锁
- **多任务模糊** - 切换到多任务视图时，自动模糊屏幕内容，防止他人窥探

## 开启应用锁

1. 进入「我的」→「个性化设置」
2. 开启「应用锁」
3. 设置 4 位数字 PIN 码
4. 可选开启生物识别（指纹/面容）

## 锁定时机

- **应用启动** - 每次打开应用时需要验证
- **从后台返回** - 切换到其他应用后再回来时需要验证
- **超时设置** - 可设置超时时间（立即、1 分钟、5 分钟、15 分钟）

## 隐私模糊屏

开启应用锁后，当你切换到多任务视图（上滑查看最近使用的应用）时：

- 应用内容会自动模糊处理
- 其他人无法在多任务视图中看到你的财务数据
- 返回应用后模糊自动消失

:::tip 提示
隐私模糊屏仅在应用锁开启时生效。如果不需要应用锁但希望保护多任务隐私，也需要先开启应用锁。
:::

## 忘记 PIN 码

如果忘记了 PIN 码，可以通过以下方式重置：

1. 卸载并重新安装应用
2. 重新设置应用锁

:::warning 注意
重置前请确保数据已通过云同步备份，否则本地数据可能丢失。
:::

## Web 端二次验证(2FA)

[BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端支持 TOTP 二次验证 — 用 Google Authenticator / 1Password / Authy 等扫码,登录时除了密码再填 6 位动态码。

### 开启 2FA

1. 登录 Web 端 → 头像菜单 → **个人资料**
2. 顶部「二次验证」chip 点击 → 弹出引导:
3. 用 Authenticator App 扫描二维码(或手输 secret)
4. 输入当前 6 位动态码确认
5. 保存 8 个一次性 recovery codes(打印 / 存密码管理器,设备丢失时救命)

开启后再登录会多一步:**密码 → 6 位动态码**。Web 端首次登录设备会"信任 30 天"(可在设备管理里手动撤销)。

### 关闭 2FA

个人资料页 → 二次验证 chip → 输入当前动态码 → 关闭。

### 设备管理

头像菜单 → **设备**:列出所有登录过的 web / mobile 设备(IP、最后活跃时间、客户端类型),可一键吊销异常设备的访问 token,被踢的设备下次请求自动跳回登录页。

> 说明:手机端 App 当前不展示 2FA 选项,二次验证策略由 Web 端管理,生效范围覆盖所有登录入口。
