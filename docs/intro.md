# 欢迎使用蜜蜂记账

蜜蜂记账是一款**开源免费**的个人记账应用，支持 iOS 和 Android 双平台。

## 为什么选择蜜蜂记账？

- **简洁易用** - 极简设计，快速记账，不打断你的生活节奏
- **数据安全** - 数据完全存储在本地，支持多种云同步方式，数据掌控在自己手中
- **功能完善** - 多账本、多账户、分类管理、预算、标签、周期记账等
- **开源免费** - 无广告、无会员、无隐藏收费，代码完全公开
