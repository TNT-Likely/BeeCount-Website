# 收支颜色方案

自定义收入和支出的显示颜色，适应不同地区的阅读习惯。

## 为什么需要切换颜色方案？

不同地区对颜色的理解不同：

- **红色收入·绿色支出** - 中国传统习惯，红色代表喜庆和收获
- **红色支出·绿色收入** - 国际通用习惯，红色代表警示和流出

## 修改颜色方案

1. 进入「我的」→「个性化设置」→「个性装扮」
2. 点击「收支颜色方案」
3. 选择你偏好的方案

## 颜色方案选项

### 红色收入·绿色支出（默认）

- ✅ 收入显示为红色
- ✅ 支出显示为绿色
- 适合习惯中国传统的用户

### 红色支出·绿色收入

- ✅ 支出显示为红色
- ✅ 收入显示为绿色
- 适合习惯国际标准的用户

## 应用范围

颜色方案会应用到：

- 首页收支统计
- 交易列表金额
- 账户余额显示
- 统计图表
- 日历视图
- 分类详情
- 标签详情

## 实时生效

选择后立即生效，无需重启应用，所有页面的收支颜色都会自动更新。

## 搭配主题色

收支颜色方案与主题色独立设置：

- 主题色影响导航栏、按钮等界面元素
- 收支颜色专门影响金额数字显示
- 两者可以自由组合，打造个性化界面

## Web 端配色 / 头像 / 昵称

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端 → 头像菜单 → **个人资料**:

- **头像** — 点击头像 → 选图上传(≤4 MB),通过云同步推到手机端
- **昵称** — 点击名字 inline 编辑,Enter 保存 / Esc 取消
- **收支配色** — 「同步偏好」卡片里点击切换,跟手机端双向实时同步
- **暗黑模式头部图案** — Select 4 选项(无装饰 / 图标平铺 / 粒子星星 / 蜂巢六边形)
- **余额显示格式** — Select(完整金额 / 简洁显示)
- **显示交易时间** — iOS 风 Switch 切换

任意端改完通过 `profile_change` WS 实时下发到对端。
