# 分类管理

分类帮助你归类收支，便于统计分析。

![分类管理](/img/preview/zh/09-category-management.png)

## 预设分类

应用内置常用分类：

**支出**：餐饮、交通、购物、娱乐、居住、通讯、医疗、教育...

**收入**：工资、奖金、投资收益、兼职、红包...

## 添加分类

1. 进入「我的」→「数据管理」→「分类管理」
2. 点击右上角「+」
3. 选择图标
4. 输入分类名称
5. 保存

## 编辑分类

点击分类进入编辑页面，可修改：

- 分类名称
- 图标（支持内置图标和自定义图片）

## 自定义分类图标

除了内置的图标库，你还可以为分类设置自定义图片：

1. 进入分类编辑页面
2. 点击图标选择区域
3. 选择「自定义图标」
4. 从相册选择图片

### 自定义图标特性

- **自动裁剪** - 图片会自动裁剪为正方形
- **云端同步** - 如果启用了云同步，自定义图标会自动同步到云端
- **配置导出** - 导出配置时会包含自定义图标数据
- **附件支持** - 导出附件时会包含自定义图标图片文件

### 使用建议

自定义图标适合以下场景：

- 商家品牌 Logo（如星巴克、麦当劳）
- 特殊分类（如宠物、爱好）
- 更直观的视觉识别
- 个性化记账体验

## 排序分类

长按分类可拖拽排序，将常用分类放在前面，提高记账效率。

## 删除分类

删除分类时，可以选择将该分类下的交易迁移到其他分类。

![分类迁移](/img/preview/zh/08-category-migration.png)
