# 更新日志

记录蜜蜂记账的版本更新历史。

## v2.2.0 (2024-12)

### 新功能

- ✨ **暗黑模式** - 支持深色主题，OLED 友好
- ✨ **标签系统** - 为交易添加多个标签
- ✨ **预算管理** - 设置月度预算和分类预算
- ✨ **周期记账** - 自动记录固定收支
- ✨ **发现页** - 新增功能入口集合页

### 改进

- 🎨 全面适配暗黑模式
- 🎨 优化 UI 细节
- ⚡️ 性能优化

## v2.0.0 (2024-12)

### 重大更新

- 🔄 **账户独立改造** - 每个账户独立追踪余额
- 📂 **二级分类** - 支持父子分类结构
- ☁️ **iCloud 同步** - iOS 用户零配置同步
- 🪣 **S3 协议支持** - 支持 Cloudflare R2、AWS S3 等

### 新功能

- ✨ 账户详情页
- ✨ 账户间转账
- ✨ 分类迁移功能
- ✨ 配置导入导出
