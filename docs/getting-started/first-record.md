# 第一笔记账

本指南将帮助你快速完成第一笔记账。

## 打开记账页面

点击底部中央的 **「+」按钮** 进入记账页面。

![记账页面](/img/preview/zh/03-edit-transaction.png)

## 记录一笔支出

1. **选择类型** - 默认为「支出」，也可以切换到「收入」或「转账」
2. **输入金额** - 使用数字键盘输入金额
3. **选择分类** - 点击分类图标选择消费类别
4. **选择账户** - 选择支出的账户（如：现金、银行卡）
5. **添加备注**（可选）- 点击备注输入框添加说明
6. **完成** - 点击右上角「完成」按钮保存

## 快捷操作

### 修改日期时间

点击日期时间区域可以修改记账的日期和时间，方便补记之前的消费。

### 添加标签

点击「添加标签」可以为这笔交易打上标签，方便后续筛选和统计。

### 添加图片

点击相机图标可以拍照或从相册选择图片，比如保存消费凭证。

## 查看记录

返回首页即可看到刚才记录的交易，按日期分组展示。

- **点击** 记录可以编辑
- **左滑** 记录可以删除
