# 记账功能

蜜蜂记账支持三种交易类型：支出、收入和转账。

## 支出记录

记录日常消费支出，会从选定账户中扣减余额。

## 收入记录

记录工资、奖金等收入，会增加选定账户的余额。

## 转账记录

在不同账户之间转移资金，如从银行卡取现到现金账户。

- 转出账户余额减少
- 转入账户余额增加

## 交易详情

每笔交易可以包含：

- **金额** - 必填，交易金额
- **分类** - 必填，消费类别
- **账户** - 必填，关联的账户
- **日期时间** - 默认当前时间，可修改
- **备注** - 可选，交易说明
- **标签** - 可选，多个标签
- **图片** - 可选，消费凭证

## Web 端交易管理

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端 → 顶部导航 **交易**:

### 列表与筛选

- **顶部搜索栏** — 关键词全文搜索 + 筛选弹窗(类型 / 账户 / 分类 / 标签 / 日期范围 / 金额区间)
- **桌面表格视图** — 比手机列表信息密度高;有附件 / 标签的行直接显示 chip
- **分页** — 默认 50 笔/页,可调

### 批量操作(桌面端独占)

点击右上角的 **批量选择** 图标进入选择模式:

- **行首 checkbox** — 单选 / 全选当前页
- **⇧ + Click** 范围选(Gmail 风格);**⌘ + Click** 增量选
- **批量删除** — 二次确认弹窗(显示 N 项 + 合计金额),整体回滚
- **批量导出 CSV** — 选中行下载,一次最多 200 行
- **Esc 退出选择模式**

### 全局快捷

- **⌘K / Ctrl+K** — 任意页打开命令面板,搜索 / 跳转 / 新建 / 导出 / 导入 / AI 记账
- **⌘V 粘贴图片或文字** — 命令面板下默认动作切到「AI 记账」,详见 [AI 图片识别](../ai/image.md#web-端图片记账) / [AI 文字记账](../ai/chat.md#web-端文字记账)

### CSV 导出

交易列表右上角 **「导出 CSV」** 按钮 — 按当前 filter 导出全部命中行;格式跟手机端 / mobile import 严格对齐(11 列、本地化表头、UTF-8 BOM,大文件流式下载内存稳定)。
