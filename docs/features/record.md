# 记账功能

蜜蜂记账支持三种交易类型：支出、收入和转账。

## 支出记录

记录日常消费支出，会从选定账户中扣减余额。

## 收入记录

记录工资、奖金等收入，会增加选定账户的余额。

## 转账记录

在不同账户之间转移资金，如从银行卡取现到现金账户。

- 转出账户余额减少
- 转入账户余额增加

## 交易详情

每笔交易可以包含：

- **金额** - 必填，交易金额
- **分类** - 必填，消费类别
- **账户** - 必填，关联的账户
- **日期时间** - 默认当前时间，可修改
- **备注** - 可选，交易说明
- **标签** - 可选，多个标签
- **图片** - 可选，消费凭证
