# 周期记账

自动记录固定收支，适合工资、房租、订阅等周期性交易。

## 创建周期账单

1. 进入「发现」→「周期记账」
2. 点击「+」创建新周期账单
3. 填写交易信息（金额、分类、账户等）
4. 设置周期类型和执行时间

## 周期类型

- **每日** - 每天自动记录
- **每周** - 每周固定日期
- **每月** - 每月固定日期
- **每年** - 每年固定日期

## 使用场景

- **工资** - 每月固定日期收入
- **房租** - 每月固定支出
- **订阅服务** - 如视频会员、音乐会员
- **话费** - 每月固定扣费

## 管理周期账单

- 可以暂停、恢复或删除周期账单
- 支持设置开始日期和结束日期
