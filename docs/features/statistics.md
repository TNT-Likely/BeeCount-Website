# 统计分析

蜜蜂记账提供丰富的统计图表，帮助你了解自己的收支情况。

![统计页面](/img/preview/zh/04-chart-analysis.png)

## 收支趋势

折线图展示每日/每周/每月的收支变化趋势，直观了解消费规律。

## 分类占比

饼图展示各分类的支出占比，找出你的主要消费方向。

## 分类排行

列表形式展示分类排行，点击可查看该分类下的所有交易。

## 时间筛选

支持按周、月、年查看统计数据，左右滑动切换时间段。

## Web 端统计

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端后,桌面浏览器有更强的可视化能力:

- **首页 Hero** — 当月总览 + 收支曲线 + 资产构成饼图 + Top 分类 / 标签 / 账户
- **概览页** — 多月对比、收入支出双线图、本月/上月差异
- **日历视图** — 月历方格按天显示净值,点击进交易明细
- **年度报告** — 12 屏沉浸式回顾,生成可分享的海报(适合年终复盘 + 朋友圈分享)
- **分析页** — 跨月趋势 + 分类深度分析

所有数据跟手机端共用同一份云端账本,分析口径完全一致。
