# 预算管理

设置预算帮助你控制支出，养成良好的消费习惯。

## 进入预算管理

每个账本独立设置预算。3.2.1 起,预算管理入口挪到账本管理菜单内:

1. 进入「我的」→「账本管理」
2. **长按**目标账本(任意 member 都可看,Owner 可编辑)
3. 选择「预算管理」

> 历史路径(3.2.0 及之前):「我的」→「预算管理」一级入口。

## 总预算

设置每月的总支出预算,实时追踪支出进度。

1. 在预算管理页点击「设置总预算」
2. 输入月度预算金额

## 分类预算

为特定分类设置预算，如：

- 餐饮：2000 元/月
- 娱乐：500 元/月
- 购物：1000 元/月

## 预算提醒

- 支出达到预算的 80% 时提醒
- 超支时显示警告

## 预算周期

预算按自然月计算，每月 1 日自动重置。
