# 净值趋势(net worth over time)

> ✨ **v3.5.0 新增**;自建云用户需要 BeeCount Cloud ≥ 1.5.0。

光看当下的净资产数字,看不出"这半年是攒下了还是花出去了"。净值趋势把你的**净资产随时间的变化**画成一条曲线,一眼看清财务走向。

## 在哪看

打开资产页,顶部图表区支持**走势 / 资产构成**一键切换:

- **走势**:净资产随时间变化的曲线
- **资产构成**:当前各类账户占比(原有的构成图)

点击图表可进入**全屏趋势页**,看得更清楚。

## 全屏趋势页

全屏页提供两组切换,自由组合:

- **时间范围**:近 3 个月 / 6 个月 / 12 个月 / 全部
- **指标**:净资产 / 总资产 / 总负债

可以单看负债怎么还下去的,也可以看总资产怎么涨上来的。

## 折算口径

- 净值、资产构成、趋势曲线**统一折算到[主币种](./multi-currency.md)**,多币种账户按你维护的汇率换算后再汇总
- 只有一种币种时直接用该币种,无需设置
- 缺汇率的币种账户会被排除在折算之外(同[多币种](./multi-currency.md)说明)

## 共享账本的处理

- **以成员身份加入的共享账本一律排除**:净值趋势只统计你自己的账本资产,不把你作为 Editor 加入的别人账本的资产算进来,避免重复计算和口径混乱
- 你作为 Owner 的账本正常计入

## 注意事项

- 趋势按**记账周期 / 自然日**的资产快照绘制,历史点位依据交易随时间累计计算
- 当天发生的交易会**计入当天**的净值点位(不会被时间边界截断)
- 修改主币种或某币种汇率后,整条趋势曲线会按新汇率重新折算

## 版本要求

| 端 | 要求 |
|---|---|
| App | ≥ 3.5.0 |
| BeeCount Cloud(自建云) | ≥ 1.5.0(提供 net-worth-history 读端点;请先升级服务端,再升 App) |

服务端的 net-worth-history 是**纯读端点**,本版无数据库 schema 迁移。老服务端没有该端点,App 会回退为本地计算或不显示远端趋势,升级 Cloud 到 1.5.0 即可。
