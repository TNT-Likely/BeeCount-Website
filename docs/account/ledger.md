# 多账本管理

蜜蜂记账支持创建多个独立账本，适合不同场景的记账需求。

![账本管理](/img/preview/zh/05-ledger-management.png)

## 使用场景

- **日常账本** - 个人日常收支
- **旅行账本** - 某次旅行的花费
- **项目账本** - 项目支出记录
- **家庭账本** - 家庭共同开销
- **副业账本** - 副业收支分开

## 创建账本

1. 在「明细」页顶部点击账本名称，进入「账本管理」
2. 点击「+」创建账本
3. 输入账本名称
4. 可选择图标

## 切换账本

点击账本即可切换，首页显示当前账本的数据。

## 账本独立

每个账本的数据完全独立：

- 交易记录
- 统计数据
- 分类可共享或独立
- 统计周期 — 每个账本可单独设置[「每月起始日」](./month-start-day.md)(✨ v3.4.0),如个人账本对齐发薪日、信用卡账本对齐账单日

## 删除账本

删除账本会同时删除该账本下的所有交易记录，请谨慎操作。

建议先导出数据备份。

## Web 端账本管理

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端 → 头像菜单 → **账本**:

- **新建 / 编辑 / 删除** — 跟手机端等价。**首次登录还没账本时,顶部账本选择器位置自动变成「+ 新建账本」CTA**,一键创建
- **账本卡片** — 显示交易笔数 / 收入 / 支出 / 余额 / 最近更新时间;点击编辑名称 / 币种,卡片右上角 ↑ 上传图标可直接[导入数据](../record/import-export.md)到该账本
- **顶部账本选择器** — 全局切账本入口,所有列表 / 统计 / AI 记账都跟随当前账本

Web 端跟手机端共用一套账本数据,修改通过 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) 双向实时同步。
