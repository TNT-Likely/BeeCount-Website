# 预算管理

设置预算帮助你控制支出，养成良好的消费习惯。

## 总预算

设置每月的总支出预算：

1. 进入「账本管理」(「明细」页顶部点击账本名进入)
2. **长按**目标账本 → 选择「预算管理」
3. 点击「设置总预算」
4. 输入月度预算金额

## 分类预算

为特定分类设置单独预算：

| 分类 | 预算 |
|------|------|
| 餐饮 | 2000 元/月 |
| 娱乐 | 500 元/月 |
| 购物 | 1000 元/月 |

## 预算进度

首页和预算页面显示：

- 已支出金额
- 剩余预算
- 进度百分比

## 预算提醒

- **80%** - 达到预算 80% 时提醒
- **100%** - 超支时显示警告

## 预算周期

- 默认按自然月计算,每月 1 日自动重置
- 可查看历史月份预算执行情况

> ✨ **v3.4.0 起**:预算周期跟随账本的[「每月起始日」](./month-start-day.md)。把起始日设为发薪日(如 10 号),预算就按"10 号 ~ 次月 9 号"为一个周期统计与重置;Web 端预算进度同口径。

## Web 端预算管理

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端后,在头像菜单 → **预算** 页:

- **总预算 / 分类预算** — 增删改跟手机端等价,实时同步
- **首页预算条** — 桌面首页显示当前月预算执行进度,超支自动变红
- **历史月份切换** — 桌面表格视图查看历史预算执行情况更直观
