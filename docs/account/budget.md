# 预算管理

设置预算帮助你控制支出，养成良好的消费习惯。

## 总预算

设置每月的总支出预算：

1. 进入「发现」→「预算管理」
2. 点击「设置总预算」
3. 输入月度预算金额

## 分类预算

为特定分类设置单独预算：

| 分类 | 预算 |
|------|------|
| 餐饮 | 2000 元/月 |
| 娱乐 | 500 元/月 |
| 购物 | 1000 元/月 |

## 预算进度

首页和预算页面显示：

- 已支出金额
- 剩余预算
- 进度百分比

## 预算提醒

- **80%** - 达到预算 80% 时提醒
- **100%** - 超支时显示警告

## 预算周期

- 按自然月计算
- 每月 1 日自动重置
- 可查看历史月份预算执行情况
