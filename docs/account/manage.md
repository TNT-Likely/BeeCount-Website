# 账户管理

蜜蜂记账支持多账户独立管理，每个账户单独追踪余额。

## 账户类型

- **现金** - 手头现金
- **储蓄卡** - 银行储蓄账户
- **信用卡** - 信用卡账户，支持额度管理和还款提醒
- **支付宝** - 支付宝账户
- **微信** - 微信钱包
- **其他** - 其他类型账户

## 创建账户

1. 点击底部导航「资产」
2. 点击「+」添加账户
3. 选择账户类型
4. 输入名称和初始余额
5. 可选填开户行、卡号后四位、备注等信息

## 账户余额

系统自动根据交易更新余额：

- **支出** → 余额减少
- **收入** → 余额增加
- **转账** → 转出减少，转入增加

## 资产管理

资产管理页展示所有账户的净资产总览：

- **多币种支持** - 不同币种的净资产分组显示
- **资产构成图** - 直观展示各类账户占比
- **总资产/总负债** - 一目了然的财务概况

> ✨ **v3.5.0 起**:设好[主币种](./multi-currency.md)后,资产页可把多币种账户**按主币种统一折算汇总**;顶部图表支持[净值趋势](./net-worth-trend.md)与资产构成一键切换。

## 信用卡功能

信用卡账户支持额外的专属功能：

- **额度管理** - 设置信用额度，查看已用和可用额度
- **账单日/还款日** - 设置每月账单日和还款日
- **还款倒计时** - 详情页显示距还款日天数，临近时自动变色提醒
- **快捷还款** - 一键记录还款（转账到信用卡账户）

## 账户详情

点击账户查看：

- 当前余额、收入、支出统计
- 分类收支饼图分析
- 交易记录（分页加载）

## 账户排序

账户按类型分组显示（现金、储蓄卡、信用卡等），在每个分组内支持长按拖拽排序，调整你习惯的顺序。

## 默认账户

可以设置默认的收入和支出账户，记账时自动选中，提高效率。

## 隐藏账户

不再使用的账户（换掉的银行卡、注销的钱包、卖掉的车…）可以**隐藏**,而不必删除:

- **隐藏后**不会出现在记账 / 转账 / 周期记账的账户选择里,账户管理页也把它收进底部「已隐藏」分区;
- 但它的**历史交易与余额完整保留、仍计入净资产 / 资产构成 / 净值趋势**——隐藏只是让它从「选择」里退场,不改变「钱在哪」;
- 随时可**恢复**回在用状态。

在账户编辑页点「隐藏账户」即可(与删除并列);想找回就到账户管理页底部「已隐藏」分区点「恢复」。

> **为什么用隐藏而不是删除?** 删除会连账户信息一起抹掉,关联的历史交易会失去账户归属;隐藏则完整留数据、可逆。想让某账户彻底不计入资产,才用删除。
>
> 云模式下,隐藏状态跨设备同步;**Web 端可见、以及重装 App 后走全量同步不丢隐藏状态**,需服务端升级到 **BeeCount Cloud 1.6.1** 及以上(本机与双设备增量同步不受影响)。

## Web 端账户管理

登录 [BeeCount Cloud](../cloud-sync/beecount-cloud.md) Web 端 → 顶部导航**资产**:

- **创建 / 编辑 / 删除** — 跟手机端一致,通过云同步秒级推到手机
- **点击账户卡片** — 弹出该账户下交易明细 Dialog,适合快速翻账
- **批量导入自动建账户** — Web 端 [导入账本数据](../record/import-export.md) 时,文件里的账户名不存在会自动批量创建,跟手机端导入一致
- **首页资产构成饼图** — 桌面端可视化更清晰

信用卡额度 / 账单日 / 还款日 / 默认账户等深度配置目前仍在手机端管理,Web 端只读展示。
