# 账户管理

蜜蜂记账支持多账户独立管理，每个账户单独追踪余额。

## 账户类型

- **现金** - 手头现金
- **储蓄卡** - 银行储蓄账户
- **信用卡** - 信用卡账户，支持额度管理和还款提醒
- **支付宝** - 支付宝账户
- **微信** - 微信钱包
- **其他** - 其他类型账户

## 创建账户

1. 进入「我的」→「资产管理」
2. 点击「+」添加账户
3. 选择账户类型
4. 输入名称和初始余额
5. 可选填开户行、卡号后四位、备注等信息

## 账户余额

系统自动根据交易更新余额：

- **支出** → 余额减少
- **收入** → 余额增加
- **转账** → 转出减少，转入增加

## 资产管理

资产管理页展示所有账户的净资产总览：

- **多币种支持** - 不同币种的净资产分组显示
- **资产构成图** - 直观展示各类账户占比
- **总资产/总负债** - 一目了然的财务概况

## 信用卡功能

信用卡账户支持额外的专属功能：

- **额度管理** - 设置信用额度，查看已用和可用额度
- **账单日/还款日** - 设置每月账单日和还款日
- **还款倒计时** - 详情页显示距还款日天数，临近时自动变色提醒
- **快捷还款** - 一键记录还款（转账到信用卡账户）

## 账户详情

点击账户查看：

- 当前余额、收入、支出统计
- 分类收支饼图分析
- 交易记录（分页加载）

## 账户排序

账户按类型分组显示（现金、储蓄卡、信用卡等），在每个分组内支持长按拖拽排序，调整你习惯的顺序。

## 默认账户

可以设置默认的收入和支出账户，记账时自动选中，提高效率。
