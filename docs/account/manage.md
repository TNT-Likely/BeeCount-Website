# 账户管理

蜜蜂记账支持多账户独立管理，每个账户单独追踪余额。

## 账户类型

- **现金** - 手头现金
- **储蓄卡** - 银行储蓄账户
- **信用卡** - 信用卡账户
- **电子钱包** - 支付宝、微信等
- **投资账户** - 股票、基金等

## 创建账户

1. 进入「我的」→「账户管理」
2. 点击「+」添加账户
3. 选择账户类型
4. 输入名称和初始余额
5. 可选择图标和颜色

## 账户余额

系统自动根据交易更新余额：

- **支出** → 余额减少
- **收入** → 余额增加
- **转账** → 转出减少，转入增加

## 账户详情

点击账户查看：

- 当前余额
- 收支明细
- 统计图表

## 默认账户

可以设置默认的收入和支出账户，记账时自动选中，提高效率。

## 隐藏账户

不常用的账户可以隐藏，不在列表中显示，但数据保留。
