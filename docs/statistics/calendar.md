# 日历功能

日历功能让你直观查看每日收支情况，快速了解消费规律。

## 进入日历

1. 点击底部导航「统计」
2. 点击顶部「日历」标签

## 日历视图

日历以月视图展示，每个日期下方显示：

- **收入金额** - 绿色/红色显示（取决于颜色方案设置）
- **支出金额** - 红色/绿色显示（取决于颜色方案设置）
- **净收支** - 收入减支出

颜色深浅表示金额大小，一眼看出消费高峰期。

## 查看详情

点击日期可查看该日的所有交易记录：

- 按时间排序的交易列表
- 分类图标和金额
- 备注信息

## 快速导航

- **左右滑动** - 切换月份
- **快速跳转到今天** - 点击右上角「今天」按钮
- **选择月份** - 点击顶部月份标题

## 应用场景

- 📊 **消费规律分析** - 查看哪些日期消费较多
- 📅 **账单日提醒** - 标记固定支出日期
- 💰 **收入周期** - 查看工资到账日期
- 🎯 **消费控制** - 直观看到每日花费

## 与统计页面配合

日历与统计图表互补：

- 统计页面看趋势和占比
- 日历页面看具体日期的明细
- 结合使用更全面了解财务状况
