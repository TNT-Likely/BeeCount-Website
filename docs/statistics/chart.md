# 图表分析

蜜蜂记账提供多种图表，直观展示财务数据。

## 折线图

展示收支趋势变化：

- **日视图** - 每日收支金额
- **周视图** - 每周汇总
- **月视图** - 每月汇总

适合发现消费规律和异常支出。

## 饼图

展示分类占比：

- 直观显示各分类比例
- 点击扇区查看详情
- 支持收入/支出切换

适合了解消费结构。

## 分类详情

![分类详情](/img/preview/zh/07-category-detail.png)

点击分类查看：

- 该分类总金额
- 占总支出百分比
- 相关交易列表

## 数据对比

- 与上月对比
- 环比增长率
- 同比分析（年度）
