# 统计概览

蜜蜂记账提供丰富的统计功能，帮助你了解收支情况。

![统计页面](/img/preview/zh/04-chart-analysis.png)

## 首页统计

首页顶部显示当月：

- **收入** - 本月总收入
- **支出** - 本月总支出
- **结余** - 收入减支出

点击眼睛图标可隐藏/显示金额。

## 统计页面

点击底部导航「统计」进入详细统计页面：

### 收支趋势

折线图展示每日收支变化，直观了解消费规律。

### 分类占比

饼图展示各分类支出占比，找出主要消费方向。

### 分类排行

列表展示分类排行，点击可查看详情。

## 时间范围

- 左右滑动切换月份
- 支持按周、月、年查看
- 可选择自定义日期范围
